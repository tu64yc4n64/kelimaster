# Kelimaster

Türkçe kelime tahmin oyunu. Gizli kelimeyi harf harf tahmin et, renklerle yolunu bul, skorunu yükselt.

---

## Uygulama Hakkında

Kelimaster ile Türkçe kelime dağarcığını test et:

- **4–8 harf** arası kelime uzunluğu
- **Kolay, Orta, Zor, Uzman** zorluk seviyeleri
- Skor ve seri takibi
- Koyu ve aydınlık tema
- iPhone ve iPad desteği

Uygulama **App Store**'da **Kelimaster** adıyla yayındadır.
